Getting Started¶

I'm going to use the iris dataset provided through Seaborn. So, let's import the required modules and load the data.

Classification with a Random Forest¶

Pre-processing¶

Although we are fortunate in this instance to have a relatively clean dataset, we can still need to perform some preprocessing.

In [31]:
from sklearn.pipeline import Pipeline
from sklearn.compose import ColumnTransformer
from sklearn.preprocessing import LabelBinarizer, StandardScaler
from sklearn.model_selection import cross_validate, train_test_split
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import accuracy_score, multilabel_confusion_matrix

features = ['sepal_length', 'sepal_width', 'petal_length', 'petal_width']
label = 'species'

feature_trans = ColumnTransformer(
    [('standard_scaler', StandardScaler(), features)]
)

rf_classifier = RandomForestClassifier()

pipe = Pipeline(steps=[
    ('preprocessor', feature_trans),
    ('model', rf_classifier)
])

label_trans = LabelBinarizer()

Now, let's conduct a train-test split in order to create our final test data.

Aftewards, I'll train our preprocessors with the training data.

In [63]:
X_train, X_test, y_train, y_test = train_test_split(
    iris_df[features], iris_df[label],
    test_size=0.5, random_state=42
)

y_train = label_trans.fit_transform(y_train)

cv_score = cross_validate(
    pipe, X_train, y_train, cv=10, scoring='accuracy',
    return_train_score=True
)
for key in cv_score:
    print('{}: {:0.3f}'.format(key, cv_score[key].mean()))
